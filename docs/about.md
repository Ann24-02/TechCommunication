# About

File Organizer is a simple command-line tool designed to help users automatically organize files in a directory based on their file types.

The main goal of this project is to simplify file management by reducing the need for manual sorting. It scans a selected folder, detects file extensions, and moves files into corresponding directories such as images, documents, or videos.

This project was created as a demonstration of how to build and document a software tool using MkDocs and the Material for MkDocs theme. It showcases how to structure clear and user-friendly documentation, including installation instructions, usage examples, and project overview.

Although the project is a simplified example, it reflects real-world use cases and can be extended with additional features such as custom sorting rules, configuration files, or recursive directory processing.

The documentation is intended to be easy to navigate and understand, making it suitable for both beginners and more experienced users who want to quickly learn how the tool works.
